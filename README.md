# LeadsBridge – Figma Design Replication

A high-fidelity, pixel-perfect replication of the **LeadsBridge** landing page design. This project was developed as a technical assessment to demonstrate modern front-end development practices, focusing on responsiveness, clean architecture, and UI accuracy.

## 🚀 Live Demo
**[View Live Project](https://naimhossen1929.github.io/LeadsBridge/)**

---

## 🛠️ Tech Stack
* **HTML5:** Semantic markup for SEO and accessibility.
* **Tailwind CSS:** For utility-first, rapid, and responsive styling.
* **JavaScript (ES6+):** For interactive components and dynamic UI logic.
* **Google Fonts:** Integrated for precise typography, matching the Figma design.

---

## ✨ Key Features
* **Pixel-Perfect Implementation:** Closely followed the Figma design specifications (spacing, typography, and color palette).
* **Fully Responsive:** Optimized for all screen sizes, from mobile devices to large desktops.
* **Clean Codebase:** Organized file structure and reusable CSS utility classes for maintainability.
* **Interactive UI:** Smooth transitions and hover effects to enhance the user experience.
